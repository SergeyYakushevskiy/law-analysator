# Анализатор НПА

## Описание

**Анализатор НПА** — приложение для отслеживания изменений в редакциях нормативно-правовых актов.

Позволяет:

* сравнивать разные версии документов
* автоматически отслеживать изменения в папке с файлами
* визуализировать различия
* формировать статистику изменений
* экспортировать отчёты

**Версия:** v0.1.0 alpha

---

# Установка

## Требования

* Операционная система: **Windows 11**
* Дополнительные зависимости не требуются

---

## Установка

1. Перейдите в раздел **Releases**
2. Скачайте установочный файл:

   ```
   LawAnalyzerSetup.exe
   ```
3. Запустите установщик
4. Выберите директорию установки (по умолчанию: `C:\Program Files\...`)
5. Дождитесь завершения установки
6. При необходимости создайте ярлык на рабочем столе (опция в установщике)

После установки:

* приложение доступно через меню «Пуск»
* также создаётся ярлык на рабочем столе

---

# Запуск

Запустите приложение:

* через ярлык на рабочем столе
  или
* через меню «Пуск»

---

# Использование

## Создание или открытие проекта

При первом запуске необходимо:

1. Выбрать папку:

   * пустую папку
     или
   * папку с файлами `.txt`

2. Приложение:

   * инициализирует проект
   * запоминает путь к проекту
   * отслеживает изменения в файловой системе

---

## Работа с версиями документов

После загрузки папки:

* пользователь вручную задаёт **хронологический порядок редакций**
* приложение сохраняет порядок и использует его для анализа

---

## Автоматическое обновление

Приложение отслеживает изменения в выбранной папке:

* добавление файлов
* удаление файлов
* изменение содержимого

Все изменения автоматически отображаются в интерфейсе

---

## Панель сравнения документов

Основная область интерфейса разделена на два документа:

* левый документ — одна версия
* правый документ — другая версия

### Подсветка изменений:

* красный — удалённые элементы
* жёлтый — изменённые элементы
* зелёный — добавленные элементы

---

## Панель статистики

Панель статистики работает автоматически на основе выбранных файлов.

Пользователь может выбрать уровень детализации:

* пункт
* статья
* глава
* и другие структурные уровни

### Включает:

* дерево изменений (от корня до выбранного уровня)
* гистограмму частоты изменений

---

## Экспорт отчёта

Пользователь может сформировать текстовый отчёт:

* нажать кнопку **"Отчёт"**
* получить текстовый вариант статистического анализа

---

# Структура проекта

После установки приложение работает с локальными файлами проекта:

* `.txt` файлы — редакции документов
* внутренняя информация о порядке версий сохраняется приложением
* путь к последнему открытому проекту запоминается автоматически

---

# Примечания

* Приложение ориентировано на работу с текстовыми файлами `.txt`
* Для корректной работы рекомендуется не изменять файлы вне приложения без необходимости
* Изменения в папке автоматически подхватываются системой мониторинга

---

# Возможные проблемы

**1. Приложение не видит файлы**

* убедитесь, что файлы имеют расширение `.txt`
* проверьте, что выбрана правильная папка проекта

**2. Изменения не отображаются**

* убедитесь, что изменения сохранены в файле
* попробуйте повторно выбрать папку проекта
